HTML Notes

HTML Basics

HTML stands for HyperText Markup Language.

It is used to structure content on the web.

HTML Tags

HTML documents are made up of HTML tags.

Tags are used to define the structure and appearance of content.

HTML Elements

HTML elements consist of an opening tag, content, and a closing tag.

Example: <p>This is a paragraph.</p>

Text Color

You can set the text color using the class attribute.
